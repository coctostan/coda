    ```
    ]
    ```

    **Interface `ValidateContext`:**

    - `questions: readonly string[]` — VALIDATE_QUESTIONS

    - Read gap analysis via readGapAnalysis
    - Read evidence via readAllEvidence, extract module names
    - Return ValidateContext

    **── ORIENT ──**

    **Const `ORIENT_QUESTIONS: readonly string[]`:**
    ```
    [
      'Where do you want to take this project?',
      'What is the biggest pain point or opportunity?',
      'What should we NOT change? (stability boundaries)',
      'What are the roadmap priorities?',
    ]
    ```

    **Interface `OrientContext`:**
    - `gapAnalysis: { body: string } | null`
    - `evidenceSummary: string[]`
    - `questions: readonly string[]` — ORIENT_QUESTIONS

    **Function `assembleOrientContext(codaRoot: string): OrientContext`:**
    - Same evidence + gap as validate, different questions
