After SCAN gathers evidence, SYNTHESIZE tells the agent to distill that evidence into structured reference docs. These draft docs become the project's knowledge base for all subsequent CODA operations.

- `SYNTHESIZE_REF_DOCS` const describing the ref docs to produce
- `SynthesizeContext` interface

<task type="auto">
  <name>Task 1: Add SYNTHESIZE orchestration to brownfield.ts</name>
  <files>packages/coda/src/forge/brownfield.ts, packages/coda/src/forge/index.ts</files>
  <action>
    Add to the existing `brownfield.ts`:

    **Interface `RefDocSpec`:**
    - `name: string` — filename (e.g., 'ref-system.md')
    - `title: string` — human-readable title
    - `description: string` — what this ref doc covers
    - `sourceEvidence: string[]` — which evidence modules contribute (e.g., ['architecture', 'security'])

    **Const `SYNTHESIZE_REF_DOCS: readonly RefDocSpec[]`:**
    ```
    [
      { name: 'ref-system.md', title: 'System Reference',
        description: 'Capabilities assembled from evidence. Structured by capability, not by module.',
        sourceEvidence: ['universal', 'architecture', 'security', 'tdd', 'quality'] },
      { name: 'ref-architecture.md', title: 'Architecture Reference',
        description: 'Patterns, layers, data flow, module boundaries.',
        sourceEvidence: ['architecture', 'quality'] },
      { name: 'ref-conventions.md', title: 'Conventions Reference',
        description: 'Testing patterns, code style, security practices.',
        sourceEvidence: ['quality', 'security', 'tdd'] },
      { name: 'ref-prd.md', title: 'Product Requirements',
        description: 'Why the project exists, who uses it, what it does.',
        sourceEvidence: ['universal', 'knowledge'] },
    ]
    ```

    **Interface `SynthesizeContext`:**
    - `evidence: Array<{ module: string; body: string }>` — evidence content from SCAN
    - `refDocs: readonly RefDocSpec[]` — which ref docs to produce
    - `evidenceCount: number` — how many evidence files were found

    **Function `assembleSynthesizeContext(codaRoot: string): SynthesizeContext`:**
    - Call `readAllEvidence(codaRoot)` to load all evidence files
    - Map to `{ module, body }` pairs
    - Return SynthesizeContext with evidence, refDocs, and evidenceCount

    Import `readAllEvidence` from `./evidence`.
    Export new types and functions from `forge/index.ts`.

    Avoid:
    - Actually producing the ref docs (agent does that)
    - Modifying existing SCAN functions
    - Adding external dependencies
  </action>
  <verify>bun run tsc --noEmit</verify>
  <done>Functions compile, exported. AC-1 through AC-4 structurally satisfied.</done>
